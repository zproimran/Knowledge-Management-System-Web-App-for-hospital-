📌 Project Overview

The Knowledge Management System (KMS) is a web-based application developed for Afran General Hospital to centralize, organize, manage, and share institutional knowledge across departments.

The system ensures:

Secure document storage

Structured knowledge categorization

Role-based access control

Easy search and retrieval

Department-level knowledge sharing

Reduced knowledge loss

This platform helps improve decision-making, operational efficiency, and internal communication.

🎯 Purpose of the System

Hospitals generate large amounts of:

Policies

Guidelines

Medical procedures

Administrative documents

Training materials

Reports

Without proper organization, this knowledge becomes fragmented.

The KMS solves this by:

✔ Centralizing knowledge
✔ Digitizing documentation
✔ Making documents searchable
✔ Controlling access permissions
✔ Maintaining document version control

👥 User Roles & Access Levels

The system supports role-based access control:

1️⃣ Super Admin

Manage all users

Create departments

Upload/edit/delete documents

Assign roles

View system analytics

2️⃣ Department Admin

Manage department documents

Upload knowledge resources

Approve submissions

Manage department users

3️⃣ Staff/User

View documents

Search knowledge

Download files

Submit knowledge (if permitted)
